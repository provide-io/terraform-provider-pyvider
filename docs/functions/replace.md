# replace (Function)

Terraform function for replace

## Example Usage

replace("hello test", "test", "world") # Returns: "hello world"

## Signature

``replace(str, old, new)``

## Arguments

- `str`: The input string
- `old`: The substring to replace
- `new`: The replacement string
